Auto Clip YouTube → TikTok/IG dengan AI Lokal

Tempel link YouTube, pilih kualitas (Lite/High/Highest), toggle subtitle, klik render. Semua proses lokal, biaya API ditanggung user (BYO API key).

Fitur utama
  • Deteksi hook otomatis (Groq) + rencana 10–20 klip
  • Mode manual: isi rentang waktu sendiri, subtitle opsional (butuh Groq key)
  • Render FFmpeg native (Lite/High/Highest), toggle subtitle
  • Unduh YouTube dengan yt-dlp (cookies suported)
  • Cache video lokal, tanpa server berat, tanpa login
  • Hasil siap unduh (mp4) + thumbnail

Download & Lisensi

Download installer desktop Windows (EXE). Masukkan license key (jika diminta) setelah pembelian. Tidak perlu akun, tidak perlu server. Biaya API Groq ditanggung user (BYO key).

Paket

License sekali bayar, update minor gratis.

Kebutuhan
  • API key Groq (BYO)
  • cookies.txt (opsional, untuk video protected)
  • Internet + storage lokal

Cara Pakai Singkat

  1. Buka app; terminal akan ikut terbuka. Jangan tutup terminal, bisa dipakai melihat proses.
  2. Masukkan API key Groq jika ingin otomatis atau subtitle.
  3. Opsional: pilih cookies.txt untuk YouTube yang butuh login / kualitas >1080.
  4. Tempel link YouTube, pilih kualitas (Lite/High/Highest).
  5. Pilih mode:
    • Groq otomatis: transkrip + deteksi hook → 10–20 klip siap render.
    • Manual: isi rentang klip; centang subtitle jika mau transkrip (butuh Groq key).
  6. Render per klip atau Render All → unduh mp4 (atau ZIP jika banyak klip).
