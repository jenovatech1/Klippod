Panduan KlipPod (Desktop BYO Key)

1) Prasyarat

  • API key Groq (Whisper & chat) — user isi sendiri.
  • cookies.txt jika perlu download YouTube yang butuh login.
  • Internet + ruang disk lokal.
  • FFmpeg dan yt-dlp sudah dibundel/auto-download oleh app.

2) Instalasi

  1. Unduh installer EXE dari link yang diberikan setelah membeli.
  2. Jalankan, dan biarkan jendela terminal yang ikut terbuka tetap hidup (tempat melihat log/proses).
  3. Masukkan license key (jika diminta) di layar awal.

3) Konfigurasi API & cookies

  1. Buka menu Settings → tempel API key Groq.
  2. (Opsional) Pilih file cookies.txt (atau cookies.txt.txt) untuk yt-dlp.
  3. Simpan. App akan menyimpan secara lokal/terenkripsi di device.

Cara mendapatkan API key Groq (BYO):

  1. Buka https://console.groq.com.
  2. Daftar atau login (pakai email/SSO yang disediakan).
  3. Masuk ke menu API Keys → klik Create API Key → salin key (format: gsk_...).
  4. Simpan key itu ke Settings KlipPod. Jangan dibagikan ke orang lain.

4) Workflow harian

  1. Tempel link YouTube.
  2. Pilih kualitas: Lite / High / Highest, toggle subtitle on/off.
  3. Pilih mode:
    • Groq otomatis: transkrip + hook otomatis → 10–20 klip siap render.
    • Manual: isi rentang klip sendiri. Jika butuh subtitle, centang subtitle dan pastikan API key Groq terisi; transkrip akan diambil saat generate/render.
  4. Klik Generate (untuk Groq otomatis) atau langsung isi klip manual.
  5. Pilih klip yang mau dirender → Render atau Render All.
  6. Download mp4 dari kartu klip (atau ZIP jika tersedia).

5) Estimasi & batasan

  • Estimasi ukuran tampil di header (berdasar bitrate & durasi total klip terpilih).
  • Jika "Highest" tetap 1080p, kemungkinan:
    • Video YouTube hanya punya 1080p.
    • Butuh cookies untuk stream >1080.
  • Rate limit Groq: jika muncul "Too Many Requests", tunggu 1–2 menit dan coba lagi.

6) Troubleshooting ringkas

  • Gagal unduh YouTube: pastikan cookies valid; coba ulang dengan kualitas lebih rendah.
  • Subtitle tidak muncul: pastikan toggle subtitle menyala, API key Groq valid, dan transkrip berhasil diambil (lihat log di terminal).
  • Hook terlalu awal: sistem sudah skip sebelum suara, tapi jika tetap muncul, coba regenerate.
  • Render lambat: pilih kualitas Lite atau kurangi jumlah klip terpilih.

Semua proses lokal, biaya API ditanggung user (BYO key). Biarkan jendela terminal tetap terbuka saat aplikasi berjalan.
